# notes/

Concept notes in my own words. Written after a Tour chapter, a tricky Koan, or an Exercism pattern that needed unpacking.

## Format

- One file per topic: `<topic>.md` (e.g., `null-safety.md`, `data-classes.md`, `coroutines-basics.md`)
- 3-5 bullets minimum
- Link related notes with markdown links: `see [scope-functions](scope-functions.md)`

## Why write notes at all

Writing forces you to reorganize the idea in your own words. If you can't explain it in 3 bullets without re-reading the source, you didn't learn it.
